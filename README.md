# Gaming Leaderboard System

## Overview
This project is a web application to track, store, and display top scores in online games. Features real-time updates, user management, and an admin dashboard.

## Features
- Player registration and authentication
- Secure score submission via REST API
- Dynamic leaderboard display
- Admin controls for moderating entries

## Technologies Used
- Frontend: ReactJS
- Backend: Node.js/Express
- Database: MongoDB

## Running Locally
1. Clone the repo
2. Install dependencies (`npm install`)
3. Start backend (`npm run server`)
4. Start frontend (`npm start`)

## Contribution Workflow
- All feature work should reference a Jira task (e.g., `LEADERBOARD-5`) in the commit message.
- Use pull requests and review with the team before merging.
